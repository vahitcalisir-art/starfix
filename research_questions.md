# Research Questions and Hypotheses

## Main Research Question

**RQ:** How can an open-source Python-based sight reduction algorithm be developed and validated for accurate celestial navigation position fixing?

---

## Sub-Research Questions

**SRQ1:** What is the computational accuracy of the proposed Python-based sight reduction algorithm compared to established reference standards?

**SRQ2:** How does the algorithm perform across different celestial bodies (Sun, Moon, planets, navigational stars)?

**SRQ3:** What is the positional error magnitude when applying the algorithm to multi-body celestial fixes?

**SRQ4:** How does the computational efficiency of the Python implementation compare to traditional sight reduction methods?

---

## Hypotheses

**H1:** The proposed Python-based sight reduction algorithm achieves positional accuracy within acceptable navigational tolerances when compared to reference ephemeris data.

**H2:** The algorithm demonstrates consistent accuracy across different celestial bodies used in navigation.

**H3:** Multi-body celestial fixes computed by the algorithm reduce positional error compared to single-body observations.

**H4:** The Python-based implementation provides computational efficiency suitable for real-time navigation applications.
